# 📊 Sales & Performance Analysis
## 📌 Project Overview
This project analyzes sales data to identify trends, profitability, and key factors contributing to business losses.

## 🎯 Objectives
* Analyze sales performance across categories and regions
* Identify loss-making products and regions
* Understand the impact of discounts on profit

## 🛠️ Tools Used
* Python
* Pandas
* Seaborn
* Matplotlib

## 📊 Key Insights
* Technology category generates the highest sales
* Certain sub-categories like Tables, Binders, and Machines contribute to losses
* High discount rates significantly reduce profit
* Some regions show higher concentration of losses

## 🧠 Conclusion
The analysis shows that excessive discounting and low-margin products are the main reasons for losses. Businesses can improve profitability by optimizing pricing strategies and focusing on high-performing products.
